ronny.tw/data

 ronny.tw/data
 這幾年收集整理了很多資料，做了很多平台，但是常常到後來自己都忘了有收集過哪些資料，因此打算把做過的東西都放在這邊
 平台
 
 
 台灣公司資料
 company.g0v.ronny.tw
 資料來源為 經濟部全國商工行政服務入口網 ，更新頻率為每個月一次，包含全台灣公司名單營業項目、資本額、董監事名單，以及台灣各商號資料等
 
 台灣公司關係圖
 
 company-graph.g0v.ronny.tw
 將台灣公司資料的公司以董監事名單連結起來，只要輸入統一編號就可以看到以該公司為中心的公司關係圖

 
 關務署進出口資料整理
 
 portal.g0v.ronny.tw
 來自經濟部關務署的所有進出口貨物的進口出口國家、貨物項目申報的重量以及價值資料，以月為單位，並且有畫成圖表可以看出歷史的變化，資料從 2003 年開始，更新頻率為每個月

 
 PTT熱門記錄
 
 ptthot.ronny.tw
 從 2011/11 開始收集每五分鐘抓取PTT熱門看板人氣數字，並記錄當時的板名，可以看出 PTT 各看板在各社會事件的人氣變化

 
 NewDiff
 
 newsdiff.g0v.ronny.tw
 從 2013/8 開始收集各網路媒體的新聞內容，並且追蹤是否有被刪除或是內容修改，以方便研究新聞修改原因，持續自動更新中

 
 NewsTrend
 
 newstrend.g0v.ronny.tw
 利用 NewsDiff 資料做的服務，可以輸入關鍵字，搜尋該關鍵字在各媒體出現的歷史記錄，以方便分析各媒體的新聞偏好以及事件的關注度

 
 歷史四大報頭條
 
 oldpaper.g0v.ronny.tw
 收集 2011/11 開始的每日四大報頭條新聞，以方便研究時事記錄以及研究媒體偏好

 
 台電發電量歷史記錄
 
 taipower.g0v.ronny.tw
 每十分鐘收集台電各種發電量數字，畫出歷史趨勢圖

 
 台北市建築執照記錄
 
 tpebuilding.g0v.ronny.tw
 收集台北市從民國 38 年到 102 年的建築使用執照掃描本，目前資料沒有再更新，正在處理其他來源希望可能直接針對全台灣資料做處理

 
 政治獻金切豆腐
 
 campaign-finance.g0v.ronny.tw
 利用影像處理將政治獻金的掃描圖檔抓出框線並切出每一個格子，並接力將每一格丟給 @ctiml 做的 http://campaign-finance.g0v.ctiml.tw/ 利用群眾外包來將掃描文件 OCR 出來

 
 GitHub GeoJSON 地圖產生器
 
 github.ronny.tw
 將 CSV 或是 GeoJSON 丟到 Github 之後，只需要把網址的 github.com 改成 github.ronny.tw 就可以直接瀏覽結果，因為 Github 是用 js 在瀏覽器畫出來，資料有 1MB 的限制， github.ronny.tw 是在 server 端畫好以 WMS 型式傳回來，因此資料無限制

 
 

 服務
 
 
 求職小幫手
 
 jobhelper.g0v.ronny.tw
 源起
 收集各縣市違反勞基法公司名錄，並且做成 Chrome Extension ，讓使用者安裝之後，只要在瀏覽 104, 1111, 518, yes123 等人力銀行時，只要該公司在違法名單中就會跳出警告

 
 新聞小幫手
 
 newshelper.g0v.ronny.tw
 黑客松提案投影片
 黑客松提案影片
 針對 Facebook 的瀏覽器外掛，當塗鴉牆上分享的連結被人回報是有誤的新聞時，會跳出警告這是有害新聞。當你之前看過的新聞被人回報有誤時，也會跳出警告說你前幾日看的哪則新聞剛剛被人回報，避免錯誤的資訊被流傳。

 
 斧頭幫大挑戰
 
 axe.g0v.tw

 資料整理
 
 
 
 台北市門牌整理
 Fusion Table
 從 台北市政府民政局 門牌整合檢索系統 爬下來的台北市門牌資料，一共有 27 萬個門牌和經緯度座標，這個資料是民政局人員人工測量的，因此比 Google Map 的座標更準確，座標是在門牌所在的位置上，例如那種一棟有 40 戶的社區，他的座標也是訂在各自門牌上而不是訂在馬路邊的門口

 
 台灣工廠資料
 Fusion Table
 2013/4/26 爬的工廠資料，當時有將寫爬蟲的過程實況播出，目前錄影在 YouTube 上面

 
 台北市 open data geojson 整理
 GitHub github.ronny.tw
 將 data.taipei.gov.tw 上面地理相關的資料，無論是用 Big5, UTF-8 或是 TWD67, TWD97 座標的都統一轉換成經緯度的 GeoJSON，不定期會更新

 
 台灣建築執照整理
 GitHub
 從 全國建築管理資訊系統 爬下來的全台灣建築執照名單，目前資料更新時間為 2013/12 ，之後會把這些資料做成一個定期更新的可以查詢的平台

 
 台灣地號查詢API
 地號查詢API
 將 http://easymap.land.moi.gov.tw/R02/Index 資料匯出後產生的 API

 
 台灣鄉鎮村里家戶用電量
 GitHub
 從 EcoLife清淨家園顧厝邊綠色生活網 爬下來的鄉鎮和村里用電量資料，每月自動更新一次
 
 
 台北市開放空間
 GitHub github.ronny.tw
 從 台北市建管處開放空間專區 抓取的 84年至99年間核發之建造執照涉有開放空間現已完工之建案名冊 轉換成 GeoJSON 的資料 

 
 台灣地名由來地圖
 github.ronny.tw
 從 臺灣地區地名資料 將資料依經緯度畫在地圖上面

 
 國土測繪圖資網路地圖服務地標資訊
 GitHub CSV github.ronny.tw 地圖呈現
 從 國土測繪圖資網路地圖 中抓出來的五萬多筆地標的資訊

 
 碧潭吊橋都更範圍和劃定古蹟範圍
 GitHub
 碧潭吊橋原先因為墩座被劃定為都更範圍需要移動而引發公民團體抗議，之後成功爭取到被劃定為古蹟，這邊收集了新聞稿中以及都更書提到的兩個範圍的地號，產生出了 GeoJSON 可以在地圖上比較範圍
